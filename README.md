# Vanguard Services Webpage

A clean and responsive **Services webpage** built using HTML, CSS, and JavaScript. This project showcases a professional UI layout with a focus on modern design, usability, and responsiveness across devices.

## 🚀 Features

* Responsive layout (desktop, tablet, mobile)
* Sticky navigation bar with mobile hamburger menu
* Hero banner section
* Services section with interactive cards
* Call-to-action contact banner
* Footer with links
* Smooth scrolling navigation

## 🛠️ Tech Stack

* HTML5
* CSS3 (Flexbox & Grid)
* JavaScript (DOM manipulation)
* Font Awesome (icons)
* Google Fonts (Inter, Poppins)

## 📁 Project Structure

```
├── index.html
├── style.css
├── script.js
└── assets/
```

## ⚙️ Functionality

* Mobile menu toggle using JavaScript
* Hover effects and transitions for better UX
* Fully responsive grid system for service cards
